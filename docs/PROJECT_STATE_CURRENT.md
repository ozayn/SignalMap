## 5. Database Tables

From `db.py`:

| Table | Purpose |
|-------|---------|
| **signal_points** | Time-series points: (signal_key, date, value, source, confidence, metadata). Primary key (signal_key, date). |
| **wayback_snapshot_cache** | Cached follower/subscriber snapshots: (platform, canonical_url, timestamp, followers, following, posts, archived_url, etc.). |
| **wayback_jobs** | Job metadata: job_id, platform, username, status, sample, from_year, to_year, progress, error, etc. |
| **wayback_job_snapshots** | Job results: (job_id, timestamp, archived_url, followers, etc.). FK to wayback_jobs. |
| **youtube_channel_snapshots** | Channel snapshots: channel_id, captured_at, subscriber_count, view_count, video_count. |
| **youtube_comment_snapshots** | Comment text: channel_id, video_id, comment_text, captured_at, published_at. For wordcloud/sentiment. |
| **data_updates** | Last update timestamps: (key, last_updated). Key `global_update` used by cron. |
| **oil_trade_edges** | Bilateral trade: (year, exporter, importer, value, source). |

## 6. Cron Jobs

From `cron_daily_update.py` and `services/daily_updates.py`:

| Job Key | Function | Data Fetched |
|---------|----------|--------------|
| **oil** | update_brent_prices | FRED DCOILBRENTEU. Append-only to signal_points (brent_oil_price). |
| **fx** | update_fx_usd_toman | Bonbast + rial-archive + FRED. Append-only to signal_points (usd_toman_open_market). |
| **gold** | update_gold_prices | No-op (gold is annual-only; no daily source). |
| **fx_dual** | update_dual_fx_rates | FRED XRNCUSIRA618NRUG. Append-only to signal_points (usd_irr_official). |
| **oil_production_exporters** | update_oil_production_exporters | EIA/IMF via fetch_oil_production_exporters. Writes oil_production_us, _saudi, _russia, _iran. |
| **oil_trade_network** | update_oil_trade_network | UN Comtrade. Upserts oil_trade_edges. |
| **youtube_followers** | update_youtube_channel_snapshots | YouTube Data API. Channels from YOUTUBE_DAILY_UPDATE_CHANNELS. Upserts youtube_channel_snapshots. |
